This makes use of course feedback (Excel file ) table data
Create a Database called "Feedback"
Make Database Table called "comments"
PopTable.java file is used for populating comments table using data from feedback excel file
SearchTable.java file is used to query comments on any field
InsertRow.java file is used to insert any row in comments table
DeleteRow.java file is used to delete any rows in comments table
TestyFeedbac.java file has the main method to control all the the operations above
